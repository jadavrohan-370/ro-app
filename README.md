# RO Water Purifier App UI

This is a React + Vite application implementing the UI/UX design for an RO Water Purifier Mobile App.

## Features

- **Home Dashboard**: Monitor TDS levels, filter health, and quick actions.
- **Water Quality**: View weekly TDS trends and ideal range information.
- **Filter Health**: Track individual filter life and replacement schedules.
- **Service Booking**: Schedule maintenance and service.
- **Profile**: Manage user details and settings.

## Tech Stack

- **React**: UI Library
- **Vite**: Build Tool
- **Recharts**: For data visualization (charts)
- **Lucide React**: For icons
- **Vanilla CSS**: For styling (Glassmorphism, Variables, Animations)

## Getting Started

1.  Navigate to the project directory:
    ```bash
    cd ro-app
    ```

2.  Install dependencies (if not already installed):
    ```bash
    npm install
    ```

3.  Run the development server:
    ```bash
    npm run dev
    ```

4.  Open your browser at `http://localhost:5173`.

## Design Details

- **Colors**: Royal Blue (#0052CC), Aqua (#00C2FF), Green (#13C77A).
- **Typography**: Poppins (Headings), Inter (Body).
- **Style**: Modern, Clean, Glassmorphism effects.
